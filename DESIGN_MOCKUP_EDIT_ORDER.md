## State Variations

### State 1: Initial Load - No Changes
```
- All items shown as they were
- No change badges
- "Save Changes" button disabled/hidden
- "Discard" button disabled/hidden
```

### State 2: With Changes
```
- Changed items have orange badges
- New items have green badges
- Removed items show removed state
- "Save Changes" button enabled (green)
- "Discard" button enabled (red)
- Summary shows "2 items edited, 1 added"
```

### State 3: Saving in Progress
```
- All buttons disabled
- Loading spinner on "Save Changes"
- Message: "Saving order..."
- Prevents user interaction
```

### State 4: Save Complete
```
- Success message shown
- "Save Changes" button returns to normal
- Summary clears
- Ready for more edits or back to orders
```
